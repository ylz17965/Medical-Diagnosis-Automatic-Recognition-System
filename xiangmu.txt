  3. 添加元数据过滤:

  4. 实现混合查询:

     - 使用RRF (Reciprocal Rank Fusion) 合并结果
  
  代码结构:
    ```python
    class HybridRetriever:
        def __init__(self, vector_store, keyword_index):
            self.vector_store = vector_store
            self.keyword_index = keyword_index  # 可以是简单的倒排索引
        
        def retrieve(self, query, top_k=5):
            # 向量检索
            vector_results = self.vector_store.similarity_search(query, k=top_k*2)
            # 关键词检索
            keyword_results = self.keyword_index.search(query, k=top_k*2)
            # RRF融合
            return self._rrf_fusion(vector_results, keyword_results, top_k)
    ```
输出:

    ```python
    class ConfidenceCalculator:
        def calculate(self, sources: List[Source], answer: str) -> float:
            # 因素1: 检索置信度 (来源分数的平均/加权)
            retrieval_confidence = self._aggregate_source_scores(sources)
            
            # 因素2: 来源一致性 (多个来源是否一致)
            consistency_score = self._check_source_consistency(sources)
            
            # 因素3: 生成不确定性 (若使用概率模型，可用entropy)
            generation_confidence = self._estimate_generation_confidence(answer)
            
            # 综合: 加权平均 (可调权重)
            final_score = (retrieval_confidence * 0.4 + 
                          consistency_score * 0.4 + 
                          generation_confidence * 0.2)
            return min(final_score, 0.99)  # 上限，避免100%
    ```
  
  分级显示:
    - 0.9-1.0: "高置信度" (绿色)
    - 0.7-0.9: "较高置信度" (蓝色)
    - 0.5-0.7: "仅供参考" (黄色)
    - <0.5: "建议就医" (红色)
输出:

验证:

    class PromptManager:
        def generate(self, state: DialogueState, user_message: str) -> str:
            base_prompt = "你是专业医疗助手，基于以下知识回答问题:\n{context}\n"

                stage_instruction = self._symptom_collection_prompt(state)
