## 录制要点

### 准备工作
1. 清理浏览器缓存，使用无痕模式
2. 准备测试账号和数据
3. 准备示例图片（体检报告、药盒）
4. 关闭无关应用和通知

### 录制设置
1. 使用OBS Studio录制
2. 分辨率：1920x1080
3. 帧率：30fps
4. 输出格式：MP4 (H.264)

### 后期处理

2. 添加字幕（黑体，白色，描边）
3. 添加转场效果（淡入淡出）
