## 备份页2：技术细节 - 状态机实现

### 状态转移规则

```typescript
function nextState(current: Stage, slots: Slots): Stage {
  if (current === 'symptom_collection') {
    if (slots.symptoms && slots.duration) {
      return 'disease_inquiry'
    }
  }
  if (current === 'disease_inquiry') {
    if (slots.confirmed) {
      return 'advice'
    }
  }
  return current
}
```

## 备份页3：技术细节 - 置信度算法

### 计算公式

```
置信度 = 检索置信度 × 0.4 
       + 一致性分数 × 0.4 
       + 生成置信度 × 0.2

其中：
- 检索置信度 = Σ(来源分数) / 来源数量
- 一致性分数 = 1 - (矛盾来源数 / 总来源数)
- 生成置信度 = 基于输出熵估计
```
